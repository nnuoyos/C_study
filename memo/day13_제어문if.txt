1. 조건문
-if문



항상 결과값이 참 혹은 거짓으로 나오는 식
비교연산자, 3<4 && x==3, 논리연산자
참 혹은 거짓을 의미하는 값, 0과 1이 들어갈 수 있다.

0: False
1: True


if(조건식){
실행할 문장;
}

위의 조건식의 결과에 상관없이 모두 검사

if(조건식){
실행할 문장;
}
====================
if(조건식){
실행할 문장;
}

위에 있는 조건식이 참이 아니면 순차적으로 검사
else if(조건식){
실행할 문장;
...
}


else if {
실행할 문장;
}



만약, 조건식이 거짓일 때 실행하고 싶은 명령이 있다면?

구조
if (조건식) {
	조건식이 참일 때 실행할 명령1;
}

else if(조건식2){
	조건식 1이 거짓이면서 조건식2가 참일 때 실행할 명령;
}

else{
	조건식이 거짓일 때 실행할 명령
}

===============

else는 항상 바로 위에 오는 if문의 조건식을 부정
if문의 조건식을 부정? == if문의 조건식이 거짓일 때 라는 전제하에

**if문에서 기억할 것
여러개의 if-else-if-else 문이 있을 때 else는 항상 바로 위에 있는 if문의 조건만을 부정한다!!!!
